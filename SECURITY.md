# Security

If you discover a security issue, please open a GitHub issue with minimal details or contact the maintainers privately.

Notes:
- API keys and tokens are stored in browser localStorage for this demo UI. Treat this as **non-secure**.
- For real deployments, move secrets to server-side storage and implement proper auth.

## Reporting guidelines

- Do not include real tokens/keys in reports
- Include steps to reproduce and a minimal PoC if possible
