# AgentCore OS v1.2.0 对外发布文案

这份文案用于 GitHub Release 正文、社媒发布、官网更新、微信群/社群转发与合作方同步。

这一版的核心口径不是“去造更底层的东西”，而是：

- 把现有工作流闭环做深
- 把推荐结构、任务信号和状态流转做统一
- 把桌面壳、连接器和发布链路继续做稳

不要使用的口径：

- 不要把这一版宣传成“自研操作系统”
- 不要描述成“自研数据库 / 编译器路线”
- 不要宣传成已经完整无人化
- 不要承诺 DMG / EXE 安装包已经恢复为默认分发方式

当前最稳的对外定位是：

- 本地优先 AI 工作底座
- 面向真实工作流程，不只是聊天
- 销售、客服、研究、创作四类工作流开始共享推荐结构和下一步路由
- `v1.2.0` 是把现有稳定闭环继续做深的一版

## 1. 官方发布主文案

### 1.1 正式版

**AgentCore OS v1.2.0 正式发布。**

这是一版能力深化更新。

我们没有把重心转去造新的底层系统，而是继续把已经进入主线的工作流闭环往深处做。

## 6. 对外 FAQ 口径

### Q1：这一版最大的变化是什么？

最大的变化是三点：

- 共享推荐契约扩展到主要工作流执行面
- 工作流任务和下一步动作成为更核心的状态信号
- 桌面壳、sidecar 和发布链路继续收口

### Q2：是不是转向做底层基础设施？

不是。

当前重点仍然是工作台、工作流、连接器、发布链路和知识 / 流程资产沉淀，不是去自研操作系统、数据库或编译器。

### Q3：适合谁？

适合：

- 想评估本地优先 AI 工作流的个人或团队
- 需要让推荐、任务和流程资产在多个应用间流转的业务方
- 正在验证销售、客服、研究、创作协同链路的产品团队

### Q4：安装方式有变化吗？

没有。

当前默认安装方式仍然是：

- 命令行安装
- 从源码运行

AgentCore OS `v1.2.0` 是一个本地优先、面向真实工作的 AI 工作底座深化版更新。当前版本重点把共享推荐契约、工作流任务信号、桌面壳运行时和发布链路继续收口，让销售、客服、研究和创作这条跨应用协同主线更适合作为当前默认版本。
