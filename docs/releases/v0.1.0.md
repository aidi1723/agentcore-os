# v0.1.0

Initial open-source release of **OpenClaw OS — Web Desktop UI Framework**.

## Highlights

- Desktop-style window manager (focus, z-order, minimize/restore, drag/snap, position persistence)
- Spotlight launcher (app search + command execution)
- Local-first data model (settings, tasks, drafts, publish history)
- Publishing hub with safe-by-default dispatch + BYO webhook connectors
- Local webhook connector example with receipts UI

## Notes

- This project is a framework/demo UI and is **not** a secure production system by default.
- “Auto publish” must be implemented via official APIs or approved third-party services.
