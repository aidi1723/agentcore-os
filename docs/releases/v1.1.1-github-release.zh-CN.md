# AgentCore OS v1.1.1

AgentCore OS `v1.1.1` 正式发布。

这一版是维护版，不追求继续扩功能，而是把当前稳定线里已经发现的问题收口修正，让公开发布口径、复用统计和双仓分发入口都更可靠。

## 本版重点

- 修正 `Knowledge Vault` 流程资产一键复用的计数时机，避免过早累计
- 修正公开发布文档中对 `openclaw -> runtime` 整理状态的表述
- 补齐 GitHub / CNB 双入口中文发布说明
- 全量同步当前推荐版本号到 `v1.1.1`

## 具体修复

### 1. 流程资产复用计数更准确

此前一键复用在某些路径上会先累加复用次数，再进入后续确认流程。

`v1.1.1` 已把计数时机收回到真实确认之后，让复用统计更可信，也更适合后续做资产分析。

### 2. 对外文档口径更克制

当前公开文档已明确说明：

- `openclaw` 到 `runtime` 的整理是部分迁移
- 当前仍保留兼容层
- 不再把兼容迁移写成已经完全完成

### 3. 国内外发布入口更统一

当前版本推荐安装方式仍然是：**命令行安装 / 从源码运行**

主仓库 GitHub：

```bash
git clone https://github.com/aidi1723/agentcore-os.git
cd agentcore-os
npm install
npm run dev
```

国内镜像 CNB：

```bash
git clone https://cnb.cool/aidiyangyu/agentcore-os
cd agentcore-os
npm install
npm run dev
```

推荐运行环境：

- Node.js `20+`
- npm

## 已验证

本次发布前已验证：

- `npm run lint`
- `npm run build`

## 当前边界

`v1.1.1` 仍然强调：

- 本地优先
- BYOK / API Key 驱动
- 关键输出保留人工审核边界
- 销售、客服、知识资产沉淀是当前最成熟的三条闭环

当前不建议宣传为：

- 完整无人公司
- 所有行业流程都已成熟覆盖
- 完全免人工审核
